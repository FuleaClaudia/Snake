==> To verify if node is installed 
node -v
==> To add the package.json 
npm init -y
==> To install express 
npm i express
==> To run the js file with the server for example on localhost:3000
node file.js
==> In the browser instead of sending a message we can send a file by specifying the file 
res.sendFile(__dirname + '/public/index.html')
